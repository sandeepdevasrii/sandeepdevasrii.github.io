How to use scp (secure copy) command in linux

Copying fies between (to and from) local and remote computers running on linux can be done using scp command.

Syntax:

Copying a file From Remote Machine to Local Machine

scp username@ip_addr:/exact/path/to/file localfileName
where, username is your user name at the remote machine, ip_addr is the IP address of the remote machine.

Copying a directory (using -r option) From Remote Machine to Local Machine
scp -r username@ip_addr:/exact/path/to/dir localdir

where, username is your user name at the remote machine, ip_addr is the IP address of the remote machine.

Copying a file From Local Machine to Remote Machine
scp localfileName username@ip_addr:/exact/path/to/RemoteFile

where, username is your user name at the remote machine, ip_addr is the IP address of the remote machine.

Copying a directory (using -r option) From Local Machine to Remote Machine
scp -r localDir username@ip_addr:/exact/path/to/RemoteDir

where, username is your user name at the remote machine, ip_addr is the IP address of the remote machine.
