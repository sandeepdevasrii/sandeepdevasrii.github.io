vim configuration commands for ~/.vimrc

This page contains essential vim configuration commands which can be added in the ~/.vimrc file to configure vim. Once these commands are added to the ~/.vimrc file, they will take effect from the vim startup. If you dont want these options to be permanently set from the startup, these commands can be issued separately after loading vim.

Remap Esc by the key sequence jk

This option will remap the key sequence jk to Esc so that reaching the top left corner of the keyboard each time for quitting the insert mode can be avoided.
Command:
inoremap jk

Fix the right margin (text width)

This option will limit the text to 'n' characters, where 'n' depends on your choice. While writing paragraphs with long sentences, a line will automatically wrap (a new line character will be inserted) to a new line if the number of characters in the line exeeds the count 'n'.
Command:
set textwidth=n
Example:
set textwidth=130

Enabling Command Line Completion

The following commands will enable command line completion for vim commands and file paths/locations.
Command:
set wildmenu
set wildmode=list:longest

Enable Spell Check for All Files Opened in Vim

The following command will enable spell check for all types of files opened in vim. If you want the spell check to be enabled only for some types of files (e.g., TeX files), it is better to use this command from the vim command prompt rather than adding it in the ~/.vimrc file.
Command:
set spell
