fim image viewer and fimgs pdf viewer

fim and fimgs are respectively the image viewer and pdf reader programs which can be used directly from a linux console like tty1. Most of the image viewers and pdf rendering tools are gui based, and hence require some graphical Desktop environment for running these tools. For example, if you try running the commonly used Evnice pdf rendering tool from tty11, you will get an error message "Cannot open display", which is obvious as tty1 is just a linux console without X desktop environment, which cannot run X based programs. However, fim and fimgs are two tools which can run from a linux console and still can display images or render pdf files directly on the linux console..!! fim and fimgs are the improved versions of fib and fbgs utilities available on most (at least on ubuntu) of the linux repos which are functionality same as fim and fimgs. Unlike fib and fbgs, fim and fimgs have several improved features like vim like keybindings etc. If you are addicted to using only the linux console, these little utilities might be of great help. For example, if you use LateX to typeset a pdf document using vim editor, you may like to use the tty1 for doing the same, and you might want to compile the LateX source and see the output. If you use these tools, you don't have to switch back to the graphical Desktop environment to see the generated pdf file. Similary, if you are writing an image processing code using C++ and openCV, you may prefer to open some image, process it and store the results. Using these utilities, you can directly see the processed result from the linux console rather than switching to the graphical Desktop environment and using some image viewer to see the output.
